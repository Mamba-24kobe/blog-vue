# 一、项目启动流程

## （一）安装依赖
### 1. cd serve 进入serve文件夹
```
cnpm i
```
### 2. cd web 进入web文件夹
```
cnpm i
```
### 3. cd admin 进入admin文件夹
```
cnpm i
```
## （二）本地启动项目
### 1. cd serve 进入serve文件夹
运行前, 请确认是否已启动本地mongoDB数据库服务，默认地址：mongodb://127.0.0.1:27017/
```
npm run serve
```
### 2. cd web 进入web文件夹
```
npm run serve
```
### 3. cd admin 进入admin文件夹
```
npm run serve
```
## （三）打包上线
### 1. cd web 进入web文件夹
```
npm run build
```
### 2. cd admin 进入admin文件夹
```
npm run build
```
### 1. cd serve 进入serve文件夹
```
npm run serve
```
